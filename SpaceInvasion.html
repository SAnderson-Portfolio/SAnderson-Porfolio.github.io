Space Invasion

Development Snapshot
Language: C++
Software: SFML

For my first year module, Games Programming, we were tasked with creating a 2D game application using Simple and Fast Multimedia Library (SFML). This was my first time programming in C++ and I have learnt a lot since this project was created but, I enjoyed the creation process and the final result so I have decided to include it as a portfolio piece. "Space Invasion" is a top down shooter inspired by some of the classics.
